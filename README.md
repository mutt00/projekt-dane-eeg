# Analiza danych EEG 'different doors'

## Setup

``` shell
git clone "https://github.com/mutt00/projekt-dane-eeg.git" && cd projekt-dane-eeg/

datalad clone "https://github.com/OpenNeuroDatasets/ds007647.git" data
datalad get data/sub-01

python3 main.py
```
